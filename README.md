# 📊 Simulateur d'Impact : Aide pour évaluer mon action pour l'emploi

Ce simulateur est un outil d'aide à la décision conçu pour mesurer l'efficience réelle des politiques et actions d'insertion professionnelle.

## 🔬 Fondements Méthodologiques
L'outil repose sur l'**analyse contrefactuelle**. Pour obtenir l'impact réel, nous isolons les succès qui ne sont pas directement imputables à l'action grâce à deux correcteurs :
1. **Effet d'Aubaine** : Sorties positives qui auraient eu lieu sans l'action.
2. **Biais de Sélection** : Performance liée à la sélection initiale de publics déjà aptes.

## 🚀 Fonctionnalités Clés
- **Double Vue Dynamique** : Bascule entre les indicateurs centrés sur le candidat (Revenus, État) ou l'entreprise (Productivité, Recrutement).
- **Synthèse Textuelle** : Génération d'un résumé analytique automatique basé sur les résultats.
- **Indicateurs Qualitatifs** : Intégration de l'Échelle de Rosenberg pour mesurer l'estime de soi.
- **Reporting A4** : Format paysage optimisé pour l'impression de rapports professionnels.

## 📄 Installation
1. Copiez les fichiers `index.html`, `style.css` et `script.js` dans un même répertoire.
2. Assurez-vous d'avoir une connexion internet pour charger la bibliothèque `Chart.js`.
3. Ajoutez un fichier `favicon.png` pour l'identité visuelle de l'onglet.
